# 📊 Análise de Vendas de Supermercado

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar dados de vendas de um supermercado utilizando Python, identificando padrões de consumo e gerando insights de negócio a partir dos dados.

A análise foi realizada utilizando técnicas de manipulação e visualização de dados, permitindo identificar os produtos mais rentáveis, as cidades com maior faturamento e os métodos de pagamento preferidos pelos clientes.

## 🚀 Tecnologias Utilizadas

* Python
* Pandas
* Matplotlib
* Git
* GitHub
